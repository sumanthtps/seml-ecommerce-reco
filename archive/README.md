# Archive

These files are the **original assignment submission and the first-pass flat
prototype**. They are preserved for reference and traceability; the live,
maintained system now lives under [`../src/reco/`](../src/reco).

| Folder / file | What it is |
|---|---|
| `legacy_prototype/` | The original single-folder prototype (`recommender_engine.py`, `recommendation_api.py`, `api_gateway.py`, demo + runner scripts). Superseded by the `src/reco` package. |
| `report_tooling/` | Scripts that generated the Word/PDF report and the submission notebook (`build_*_report.py`, `build_submission_notebook.py`, `report_evidence.py`). |
| `evidence/` | Generated console logs, metrics, plots, and the rendered report/PDF pages used as submission evidence. |
| `screenshots/` | Swagger UI and terminal screenshots used in the report. |
| `START_HERE.md`, `IMPLEMENTATION_README.md` | The original how-to guides for the flat prototype. |

> The GR4ML views (Business / Analytics Design / Data Preparation) and the
> system-architecture diagram referenced by Objective 1 are image files under
> `evidence/`.

Nothing here is imported by the live application or the test suite.
